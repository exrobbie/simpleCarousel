# simpleCarousel

## Usage

**HTML**
Create a div element
```html
<div class="enjoy"></div>
```

**JS**
Call carousel method with the element
```javascript
$('.enjoy').carousel(options);
```

## Options
```javascript
{
    imgList: [      //图片来源
        '/abc',
        '/abc'
    ],
    interval: 5000, //间隔切换时间,
    duration: 1200, //切换持续时间
}
```
